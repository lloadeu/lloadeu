**Salm 30 (29), 12-13**

// Has mudat en danses els meus planys!\
M'has tret el dol i m'has vestit de festa!//

Per això el teu Nom cantaré,\
no callaré, et lloaré per sempre.\
Només a tu cantaré:\
glòria i lloança eterna!

// Oh, Adonai! Oh Adonai!\
Déu de l’univers,\
Senyor de la creació. //

// El Cel canta l’al·leluia!\
La terra canta hosanna! \
Veient les teves meravelles. Adonai! //
